# CEO-of-your-life-_-Decision-Board-Room
Your life, run like a boardroom. Bring a real decision, hear from distinct AI directors, respond as CEO, and see how your choices evolve. Track which perspectives you engage or overrule, uncover blind spots, and get a final memo with clear tradeoffs, risks, and direction.

# Make decisions like a CEO.

This is a simple tool I built to help think through decisions more clearly. Instead of deciding on your own, you bring a decision into a “boardroom” and hear from a set of different perspectives, then respond to them as if you are the CEO.

# How it works
You enter a real decision you are facing and optional context. Five board members respond to it from different angles: risk, long-term upside, execution, values, and a contrarian view. You then respond to each of them by agreeing, disagreeing, or overruling, and they can follow up or challenge you further. At the end, the app produces a final board memo that summarizes the recommendation, tradeoffs, and downsides.

# What it tracks
Over time, the app keeps track of how you make decisions. It looks at which perspectives you agree with, which you overrule, and which you engage with the most. Patterns start to emerge, especially around which types of input you tend to ignore. That is meant to highlight potential blind spots.
How to run
Download the HTML file and open it in your browser. There is no setup required.

# API key
HIGHLY RECOMMENDED
You can paste in an OpenAI API key to get stronger and more dynamic responses. If you do not add a key, the app still works using an offline mode that generates responses based on your input, but they will be very generic.

# When to use it
This is most useful for real decisions you are actively thinking through, such as job choices, academic decisions, or any situation where you feel stuck weighing different options.
